PROMPT FUSIONADO DEFINITIVO PARA GENERAR LA APP VACKLY

Actúa como Desarrollador Senior Full-Stack experto en Expo, React Native, TypeScript estricto, NativeWind, Supabase, expo-router, i18next, UX/UI móvil premium, dashboards responsive, arquitectura SaaS moderna y producto production-ready.

Construye una app universal llamada Vackly, compatible con iOS, Android, Web y Tablet. Vackly es una aplicación integral para el control de presencia laboral y la gestión de cuidados, orientada a cuidadores profesionales, familias, administración y RRHH.

La app debe ser funcional, modular, escalable, responsive, segura, visualmente premium y fiel a la estética de las imágenes de referencia adjuntas.

IMPORTANTE SOBRE MÓDULOS DE OPINIÓN PÚBLICA:
No implementar ningún módulo de opinión pública sobre cuidadores, puntuaciones, comentarios evaluativos ni métricas públicas de reputación. No debe aparecer este tipo de funcionalidad en perfil, dashboard, base de datos, seeds, SQL, UI ni navegación.

============================================================

Construir Vackly, una app integral para:

- Registro de entradas, salidas y pausas.

============================================================
28. FLUJOS PRINCIPALES
============================================================

Flujo principal cuidador:

1. El cuidador inicia sesión.
2. Si no tiene jornada iniciada, ve botón “Iniciar turno”.
3. Desliza para iniciar.
4. Se registra entrada con timestamp, ubicación y estado NFC/geofence.
5. Si no hay conexión, se guarda offline.
6. Durante la jornada puede:
   - Iniciar pausa.
   - Finalizar pausa.
   - Rellenar diario.
   - Registrar medicación.
   - Subir gasto.
7. Al terminar, desliza para finalizar jornada.
8. Se genera resumen.
9. La familia puede validar el fichaje.
10. RRHH puede consultar si hay incidencias.

Flujo principal familia:

1. La familia inicia sesión.
2. Ve el estado actual del cuidador.
3. Consulta la persona cuidada.
4. Consulta el diario del día.
5. Revisa medicación administrada.
6. Valida fichajes pendientes.
7. Consulta tickets y gastos.
8. Invita a otros familiares con permisos diferenciados.
9. Accede a soporte si necesita ayuda.

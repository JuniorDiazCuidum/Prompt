PROMPT ÚNICO DEFINITIVO PARA GENERAR LA APP VACKLY

Actúa como Desarrollador Senior Full-Stack experto en Expo, React Native, TypeScript strict, NativeWind, Supabase, expo-router, i18next, UX/UI premium multiplataforma, arquitectura SaaS moderna, dashboards responsive y productos mobile/web production-ready.

Construye una app universal llamada Vackly, compatible con iOS, Android, Web y Tablet, orientada al control de presencia laboral de cuidadores profesionales y a la gestión de cuidados por parte de familias.

La app debe ser funcional, escalable, segura, responsive, modular y visualmente fiel a las imágenes de referencia proporcionadas.

IMPORTANTE:
- Respetar estrictamente la estética de las imágenes de referencia.
- No modificar el estilo visual.
- No rediseñar la mascota.
- No generar una UI genérica.
- No incluir módulo de reseñas ni valoraciones.
- No incluir CV/currículum del cuidador.
- No permitir contratación de cuidadores desde la app.
- Las familias no pueden contratar cuidadores desde la aplicación.
- La interfaz debe usar “Persona cuidada”.
- El backend/base de datos puede usar `patients`.
- Todas las fotos de perfil de cuidadores, familiares y persona cuidada deben ser rectangulares verticales, formato retrato aproximado 3:4, con bordes redondeados.
